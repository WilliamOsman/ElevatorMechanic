Tune the PID loop so the elevator arrives smoothly at each floor. Don't go too fast or you'll kill the passengers.  

**KP** adds motor power by how far away the elevator is from the target floor
**KI** adds motor power by how long the elevator has been away from the target floor
**KD** limits how fast the elevator can approach the target floor.
